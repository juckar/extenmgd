nueva(self, clave, cabecera='', ancho=100, siCentrado=False, siDerecha=False, rgbTexto=None, rgbFondo=None, siOrden=True, estadoOrden=0, edicion=None, siEditable=None, siMostrar=True, siChecked=False)

source code 

Contiene los mismos parámetros que la Columna.

Parameters:
  • clave - referencia de la columna.
  • cabecera - texto mostrado en el grid como cabecera.
  • ancho - anchura en pixels.
  • siCentrado - alineación
  • siDerecha - alineación, se ha diferenciado la alineación, para que al definir columnas sea más fácilmente visible el tipo de alineación, cuando no es a la izquierda.
  • rgbTexto - color del texto como un entero.
  • rgbFondo - color de fondo.
  • siOrden - si se puede ordenar por este campo
  • estadoOrden - indica cual es el orden inicial de la columna -1 Desc, 0 No, 1 Asc
  • edicion - objeto delegate usado para la edición de los campos de esta columna
  • siEditable - este parámetro se usa cuando aunque la columna tiene un delegate asociado para mostrarla, sin embargo no es editable.
  • siMostrar - si se muestra o no.
  • siChecked - si es un campo de chequeo.
Returns:
la columna creada.
